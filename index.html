GitHub for Dummies

Steps to get a new file on GitHub (* next to steps you will find yourself repeating many times over)

-hit enter/return after each step

  1. In terminal of local computer type "cd", put a space, the drag the folder the files you want to add are in. This puts you inside of that folder.
  2. Simply type "git init"
  3. Type "git add" and the file name
  4. Type "git commit -am "short message here""*
  5. Type "git branch -M main"
  6. Type "git remote add origin" and copy the link on GitHub and paste it in the terminal
  7. Type "git push -u origin main"... Anytime after that you only need to tyle "git push origin main
